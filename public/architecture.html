S/4HANA ISU Test Data Agent

AI-powered test data generation for SAP ISU scenarios

UI
💬

Chat Interface

Natural language input to describe test data scenarios. Users type prompts like "Create 10 residential customers with billing".

HTML / CSS / JS
📊

Data Viewer

Full-page overlay showing generated entities grouped by SAP table. Supports horizontal/vertical scrolling and Excel export.

SheetJS (XLSX)
↓
API
🌐

Express Server

Single API endpoint POST /api/generate receives scenario text, orchestrates the agent loop, and returns entities + messages.

Express 5 / TypeScript
📦

Entity Store

In-memory store tracking all created entities per request. Provides ID lookup for referential integrity across tools.

Per-request isolation
↓
AGENT
🤖

Agent Loop

Multi-turn orchestration loop (max 20 turns). Calls LLM, receives tool calls, executes them, feeds results back. Continues until LLM signals completion.

Agent Pattern
🧠

Azure OpenAI (GPT-4.1)

LLM with function calling. Decides which tools to invoke, generates realistic field values, and maintains entity dependency order.

OpenAI SDK / Azure
📋

System Prompt

Dynamic context builder. Injects scenario description, entity dependency rules, created entities summary, and tool call rules.
