How It Works

A 6-step agentic loop powered by Azure OpenAI function calling.

1
User gives command — e.g. "Create 5 residential customers with move-in and billing"
2
AI Agent understands the business intent and maps it to ISU entity dependency chain
3
Agent selects tools in correct order: Business Partner → Contract Account → Installation → Device → Readings → Billing
4
Tools execute with Zod-validated schemas, generating realistic IDs, names, and values
5
Validation runs automatically — checks referential integrity across all created entities
6
Data is presented in structured SAP table format with one-click Excel export
